# data and code for paper "Self-Supervised Road Layout Parsing with Graph Auto-Encoding"


## Usage
1. dataset can be found in ``datasets/``. (raw annotation of road topology is also provided in matlab format)
2. Unzip the dataset file before usage.
2. Please see the scripts in ``scripts`` for usage
